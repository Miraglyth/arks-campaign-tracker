About

This is an about page. What do you want from it?

Oh okay. This is an all-in-one tracker of campaigns in the Global server for Phantasy Star Online 2: New Genesis. It aims to consolidate all campaigns announced on the game's website, Twitter, YouTube, any other sources or simply not announced at all, and present them in a clear listing of campaigns that have ended, are presently active, and have yet to start.

It also allows users to view expanded detail for each campaign and mark them off as "Done", to help make it easy to keep track of unfinished active campaigns.

Data

This tracker does not store any data on a server or involve any user accounts. It loads a list of campaigns into your browser's storage system, which is used and updated as necessary to remember campaigns and their completion status between visits.

All saved data can be deleted by clicking the button below. Warning! This will reset all campaigns marked as completed!
